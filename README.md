# Portfolio Website

Сучасний адаптивний односторінковий сайт-портфоліо з інтерактивним калькулятором вартості розробки сайту.

## Що є на сайті

- Hero-секція з інтерактивним canvas-фоном
- Навігація по секціях і мобільне меню
- Блок послуг
- Секція кейсів/робіт
- Секція процесу роботи
- Інтерактивний калькулятор вартості
- Анімована зміна ціни
- Контактна форма
- Модальне вікно після заявки
- Адаптивний темний дизайн

## Технології

- HTML
- Tailwind CSS через CDN
- Vanilla JavaScript
- Canvas API

## Як запустити

Відкрийте файл `index.html` у браузері.

Сайт не потребує встановлення залежностей, збірки або локального сервера.

## Структура

```text
index.html   # Увесь код сайту, стилі Tailwind CDN і JavaScript
README.md    # Опис проєкту
```

## Налаштування

Основний контент можна змінити прямо в `index.html`:

- назва бренду: `Nazar.dev`
- тексти першого екрана
- послуги
- кейси
- контактні дані
- ціни в калькуляторі

Базова ціна сторінки налаштовується в JavaScript:

```js
const BASE_PAGE_PRICE = 50;
```

Додаткові послуги змінюються через значення `value` у чекбоксах.
