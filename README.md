## 💼 Бізнес-ціль проєкту
Зазвичай клієнти втрачають інтерес до сайту, якщо не бачать прозорого ціноутворення. Цей калькулятор вирішує цю проблему:
- **Динамічний розрахунок:** Ціна змінюється миттєво при виборі параметрів (кількість сторінок, додаткові опції, терміновість).
- **Зручність (UX):** Повністю адаптований під мобільні пристрої.
- **Заклик до дії:** Вбудована форма/модальне вікно для швидкого оформлення заявки після розрахунку.

---

## 🛠️ Стек технологій & Інструменти
Проєкт реалізовано за принципом швидкої розробки з використанням сучасного AI-інструментарію (Vibe Coding):
- **HTML5:** Семантична та доступна структура сторінки.
- **Tailwind CSS:** Сучасний UI-дизайн у темних тонах із плавними переходами (Transitions).
- **Vanilla JavaScript (ES6+):** Чиста логіка розрахунку вартості без використання важких сторонніх бібліотек.
- **AI-Driven Development:** Код спроєктовано та оптимізовано за допомогою штучного інтелекту, що дозволило скоротити час розробки до мінімуму без втрати якості.

---

## 📱 Функціонал
1. **Кастомізація об'єму:** Повзунок (range slider) для вибору кількості сторінок.
2. **Додаткові чекбокси:** Інтеграція послуг (Дизайн, SEO, Telegram/CRM автоматизація).
3. **Коефіцієнт терміновості:** Динамічне множення вартості при виборі термінового замовлення.
4. **Форма лідогенерації:** Модальне вікно підтвердження замовлення.
